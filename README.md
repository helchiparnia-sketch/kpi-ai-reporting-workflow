# KPI AI Reporting Workflow

## Overview
This project automates monthly KPI reporting using n8n.

## Workflow
Schedule Trigger → Google Sheets → AI Model → Email Output

## Features
- Automated monthly execution
- KPI data extraction
- AI-generated business report
- Email delivery

## Tools Used
- n8n
- OpenAI API
- Google Sheets

## Test Case
Input: February KPI data
Output: AI-generated executive report sent via email
